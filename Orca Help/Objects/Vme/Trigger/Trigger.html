Trigger Card

This card was built at the University of Washington to interface between the SNO Master Trigger Card and the NCD System (shapers and muxes).

The configuration icon looks like: It can only be placed in the VME Crate object.

Clock Record Format

version 2 (new version)
xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^^^ ^^^^ ^^-----------------------Clock record ID
-----------------^^ ^^^^ ^^^^ ^^^^ ^^^^-length (always 3 longs)
xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
-------^^-------------------------------event trigger bits
----------^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^-top-mosts bits of mac time
xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^ ^^^^-bottom 32 bits of mac time
version 1 (old version)
xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^----------------------------------Clock record ID

Global Trigger Record

The trigger card will put out the short form if at all possible, but will switch to the long form if it has to.

version 2 (new version) short form

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^---------------------------------GTID Record ID
-------^^-------------------------------event trigger bits
---------------^^^^ ^^^^ ^^^^ ^^^^ ^^^^-gtid for this trigger event

version 2 (new version) long form

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^^^ ^^^^ ^^-----------------------GTID Record ID
-----------------^^ ^^^^ ^^^^ ^^^^ ^^^^-length (always 2 longs)
xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
-------^^-------------------------------event trigger bits
---------------^^^^ ^^^^ ^^^^ ^^^^ ^^^^-gtid for this trigger event

version 1 (old version, no longer used)

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^----------------------------------Trigger ID + 1 (actually + 0x08000000)
-------^^-------------------------------event trigger bits
---------------^^^^ ^^^^ ^^^^ ^^^^ ^^^^-gtid for this trigger event
