Cable Check Task

This is a standard task that has the normal task control part. Here is the optional part:

This task will cycle thru all mux and shaper channels and verify that signals are routed to the correct locations.

The threshold values will be sent the shapers and the mux boxes for the duration of the task. The original values will be restored when the task is finished.

The width and burst rate parameters will be sent to the Pulser object for the duration of the task. The pulser's original settings will be restored when the task is finished. A pulser object must be in the configuration for this task to work.

A Pulser Distribution object must be in the configuration for this task to work.
