Mux Controller

The configuration icon for the Mux Controller is:

The Mux Controller communicates signals to and from any Mux Box connected to it as well as the High Voltage Control.

Eample of how to connect up the mux controller object:

Mux Global Data Record

The short form will be used when possible.

version 2 (new format) short form

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^---------------------------------NcdMuxBox ID
--------------------^^^^ ^^^^ ^^^^ ^^^^-global event register

version 1 (old format) long form

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^^^ ^^^^ ^^-----------------------NcdMuxBox ID
-----------------^^ ^^^^ ^^^^ ^^^^ ^^^^-length (always 8 bytes)
xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
--------------------^^^^ ^^^^ ^^^^ ^^^^-global event register

version 1 (old format)

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^-^----------------------------------NcdMuxBox ID
--------------------^^^^ ^^^^ ^^^^ ^^^^-global event register

Mux Global Data Record

The short form will be used when possible.

version 2 (new format) short form

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^---------------------------------global data record ID
-------^^ ^-----------------------------muxbox
-----------^^^ ^^^^---------------------scope bits from event register
-------------------------^^^^ ^^^^ ^^^^-chan hit register for this mux

version 2 (new format) long form

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^^^ ^^^^ ^^-----------------------global data record ID
-----------------^^ ^^^^ ^^^^ ^^^^ ^^^^-length (always 8 bytes)
xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
-------^^ ^-----------------------------muxbox
-----------^^^ ^^^^---------------------scope bits from event register
-------------------------^^^^ ^^^^ ^^^^-chan hit register for this mux

version 1 (old format)

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^----------------------------------NcdMuxBox ID + 1 (actually + 0x08000000)
-------^^ ^-----------------------------muxbox
-----------^^^ ^^^^---------------------scope bits from event register
-------------------------^^^^ ^^^^ ^^^^-chan hit register for this mux
