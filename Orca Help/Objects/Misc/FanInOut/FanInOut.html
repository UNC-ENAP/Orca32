Fan Out

Since connections can only send data or commands to one other connection point, a fan out object can be used to indirectly connect one object to many others. The configuration icon of a fan out is: A fan out can have up to ten outputs.

Fan In

Since connections can only receive data or commands from one other connection point, a fan in object can be used to indirectly connect one object to many others. The configuration icon of a fan in is: A fan in can have up to ten inputs.
