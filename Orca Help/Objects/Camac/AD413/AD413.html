AD413 CAMAC Card

This is a CAMAC card available from ORTEC. It is a four channel analog-to-digital converter with 13-bit resolution and a multiplexer. It can be used to create or use coincident signals with other cards.

Catalog Icon:

This card can exist only in a CAMAC crate.

ADC Data Format

The card will put the short form if at all possible. However, you must be prepared to decode both types.

version 2 (short form)

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^--------------------------------- AD413A ID (from header)
--------^-^^^--------------------------- Crate number
-------------^-^^^^--------------------- Card number
---------------------^^----------------- Channel number
-----------------------^-^^^^ ^^^^ ^^^^- adc value

version 2 (long form)

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^^^ ^^^^ ^^----------------------- AD413A ID (from header)
-----------------^^ ^^^^ ^^^^ ^^^^ ^^^^- length (always 2 longs)
xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
--------^-^^^--------------------------- Crate number
-------------^-^^^^--------------------- Card number
---------------------^^----------------- Channel number
-----------------------^-^^^^ ^^^^ ^^^^- adc value
