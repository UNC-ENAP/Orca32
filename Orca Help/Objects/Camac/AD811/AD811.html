AD811 CAMAC Card

This is an 8-channel analog-to-digital converter with 11-bit resolution. It is triggered by an external strobe. It is made by Ortec.

Catalog Icon:

This card can exist only in a CAMAC crate.

ADC Data Format

The card will put the short form if at all possible. However, you must be prepared to decode both types.

version 2 (short form)

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^--------------------------------- AD811 ID (from header)
--------^-^^^--------------------------- Crate number
-------------^-^^^^--------------------- Card number
-----------------------^---------------- Channel number
-------------------------^^^^ ^^^^ ^^^^- adc value

version 2 (long form)

xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
^^^^ ^^^^ ^^^^ ^^----------------------- AD811 ID (from header)
-----------------^^ ^^^^ ^^^^ ^^^^ ^^^^- length (always 2 longs)
xxxx xxxx xxxx xxxx xxxx xxxx xxxx xxxx
--------^-^^^--------------------------- Crate number
-------------^-^^^^--------------------- Card number
-----------------------^---------------- Channel number
-------------------------^^^^ ^^^^ ^^^^- adc value
